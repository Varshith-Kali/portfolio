# Varshith Kali — Portfolio

Personal portfolio website built with **Next.js**, **Three.js**, and **Framer Motion**.

## Tech Stack

- **Framework**: Next.js 16 (Static Export)
- **Styling**: Tailwind CSS v4 + shadcn/ui
- **3D**: Three.js + React Three Fiber
- **Animations**: Framer Motion
- **Language**: TypeScript

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production (generates static `out/` directory)
npm run build
```

## Deployment

This site is configured for **GitHub Pages** via Next.js static export (`output: "export"`).

After `npm run build`, the `out/` directory contains the static site ready for deployment.
